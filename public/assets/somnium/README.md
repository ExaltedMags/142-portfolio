# SOMNIUM Images

Place your SOMNIUM concert images in this folder.

## Required Images

The following image files should be placed here:
- `somnium-1.jpg` (or .png, .webp, etc.)
- `somnium-2.jpg`
- `somnium-3.jpg`

## Image Paths

These images are referenced in `src/content/achievements.ts` as:
- `/assets/somnium/somnium-1.jpg`
- `/assets/somnium/somnium-2.jpg`
- `/assets/somnium/somnium-3.jpg`

## Features

- Images are displayed in an expandable folder component
- Clicking on any image opens a full-screen lightbox
- Navigate between images using arrow keys or the navigation buttons
- Images are expandable and viewable in high resolution
